# candace-ai-agent

Candace AI Agent is a multi-project personal assistant system.

- `qq-ai-bridge/`: QQ webhook bridge, skills, memory, and vision handling
- `pc-agent/`: local desktop and browser automation runtime

Run commands:

```bash
cd qq-ai-bridge
python3 bridge.py

cd ../pc-agent
PC_AGENT_PORT=5050 python3 agent.py
```
